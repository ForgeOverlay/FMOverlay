# Forge Master Stats Overlay

Real-time item stat analysis for Forge Master with instant percentage indicators and comparisons.

## Features
- Real-time OCR stat detection
- Color-coded percentage overlays
- Item comparison tool
- Local stat tracking
- No data collection

### Install Steps
1. Download the APK file
2. Enable "Install from Unknown Sources" in Android Settings
3. Open the APK file and tap Install
4. Grant the required permissions (screen capture and display overlay)

## Permissions
- **Display Over Other Apps** - For the floating overlay
- **Screen Capture** - For analyzing game stats

## Privacy
All analysis happens locally on your device. No data is collected or transmitted.

## Usage
1. Launch the app
2. Grant screen capture permission
3. Tap the floating "SCAN" button while viewing items
4. View stat percentages with color coding
5. Use "COMPARE" to analyze item swaps

## Disclaimer
This is an independent tool not affiliated with Forge Master and doesn't interfere with the game, it just sits on top. Use in accordance with the game's terms of service.
